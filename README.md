# Light Todo Browser Extension

一个可以安装到 Chrome 或 Microsoft Edge 的轻量待办事项扩展。

## 功能

- 添加待办事项
- 标记完成或未完成
- 删除单个任务
- 按全部、未完成、已完成筛选
- 一键清空已完成
- 使用浏览器本地存储保存数据

## 安装

1. 打开 Chrome 或 Edge。
2. 进入扩展管理页：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择本项目文件夹：`H:\09-codex\ceshi-1`。
6. 点击浏览器工具栏里的 Light Todo 图标开始使用。

## 文件说明

- `manifest.json`: 浏览器扩展配置。
- `popup.html`: 扩展弹窗结构。
- `popup.css`: 弹窗样式。
- `popup.js`: 待办数据和交互逻辑。
- `icons/`: 扩展图标。
